# .glbファイルの埋め込みアニメーションの内部パス(名)の基本方針

一つの.glbファイルに複数のアニメーションをまとめて埋め込んでいる<br>
blenderでアニメーション毎に固有名をつけているのだが<br>
プログラム側で統一的に扱うために、それらのアクション-(アニメーション)固有名の対応をとる必要がある<br>
そこでその基本的方針を以下に定める


## 基本移動(行動)

### 移動
- 前へ走る: `run_fwd_default`
- 後ろ走る: `run_back_default`
- 右へ走る: `run_right_default`
- 左へ走る: `run_left_default`

### ローリング
- 前ロール: `rolling_fwd_default`
